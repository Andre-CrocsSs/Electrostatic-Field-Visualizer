# Electrostatic-Field-Visualizer
# ---English--- This program was made for the Electromagnetism I course at Facultad de Ciencias, UNAM as a final project on the course. 
## The end of this project is to watch arbitrary electrostatic charges, as well as demonstrating Coulomb's Law and the superposition principle of forces of each added charge. This code was made using the libraries NumPy and Matplotlib, they are the main requisite to run the code.
## Characteristics
- Supports multiple charges.
- Logarithmic scale to avoid visual saturation.
- Interactive visualization menu.
