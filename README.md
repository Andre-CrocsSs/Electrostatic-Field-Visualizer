# Electrostatic-Field-Visualizer
---English---
This program was made for the Electromagnetism I course at Facultad de Ciencias, UNAM as a final project on the course. The end of this project is to watch arbitrary electrostatic charges, as well as demonstrating Coulomb's Law and the superposition principle of forces of each added charge.
---Español---
Este programa fue hecho para el curso de Electromagnetismo I en la Facultad de Ciencias, UNAM como proyecto final en el curso. El fin de este proyecto es observar cargas electrostáticas arbitrarias, así como demostrar la Ley de Coulomb y el principio de superposicion de las fuerzas de cada carga.
